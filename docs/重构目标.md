# 重构目标

> 项目名：**Kairo**
> 文档版本：v1.1 · 2026-04-23
> 关联文档：[项目介绍](./项目介绍.md) · [代码规范](./代码规范.md)

---

## 一、总体目标

把现有的 **Python + Tkinter 单机版** 完全重构为 **Monorepo + React + Cloudflare Serverless** 的跨平台多用户系统，实现：

1. **多端覆盖**：Web / Windows / macOS / Linux / Android（含桌面小组件）
2. **多用户协作**：管理员可控的登录方式与访问策略
3. **Serverless 零运维**：全部跑在 Cloudflare，管理员 CLI 一键部署
4. **主题可插拔**：新功能仅通过编辑 token 文件即可完成视觉适配
5. **强一致的数据同步**：多端实时同步 + 乐观锁冲突解决
6. **安全与隐私**：敏感信息不入库，IP/地区可限流

---

## 二、不做什么（Non-Goals）

- ❌ 不迁移原 `schedule_data.json` 旧数据
- ❌ 不保留原 `scheduler.py`（重构完成后删除）
- ❌ 不上架 Google Play Store（APK 仅官网直链 + 二维码）
- ❌ 首版不做 iOS（Expo 未来可补）
- ❌ 不做端到端加密（E2EE）—— v2 再议

## 四、技术栈决定（最终稿）

| 层 | 选型 | 理由 |
| --- | --- | --- |
| 语言 | **TypeScript (strict)** | 跨端类型共享 |
| 包管理 | **pnpm workspaces** | Monorepo 标配 |
| Web | **Vite + React 18** + React Router | 轻量、纯 SPA、适配 PWA |
| 样式 | **Tailwind CSS + CSS 变量** | 主题 token 驱动，运行时切换 |
| 组件 | **shadcn/ui (Radix)** | 无障碍、可深度主题化 |
| 状态 | **Zustand + TanStack Query** | 简单 + 服务端状态 |
| 离线 | **Dexie (IndexedDB) + workbox** | 离线可写 |
| 桌面 | **Tauri 2** | 体积 < 10MB，Rust 内核 |
| 移动 | **Expo (React Native) + 原生 Kotlin Widget** | Widget 必须原生 |
| 后端 | **Cloudflare Worker + Hono + tRPC** | 类型安全 RPC |
| DB | **Cloudflare D1 (SQLite)** | 关系型 + Serverless |
| 对象存储 | **R2** | 附件/备份 |
| 缓存/Session | **KV** | 低延迟 KV |
| 队列 | **Queues + Cron Triggers** | 提醒调度 |
| 认证 | **Lucia Auth** + OAuth | 自托管、灵活 |
| 邮件 | **Email Workers / Resend** | Serverless 邮件 |
| 地图 | **MapLibre GL + Nominatim** | 默认免费开源 |
| CI/CD | **GitHub Actions + wrangler** | |
| CLI | **Node.js + Clack + wrangler API** | 与主栈统一 |

## 六、风险与缓解

| 风险 | 缓解 |
| --- | --- |
| D1 单区域延迟高 | 启用 D1 Read Replica（已公测）；读多写少 |
| Widget 开发复杂 | 先 MVP 静态显示，再加交互；参考 `react-native-android-widget` |
| 多端一致性 | 乐观锁 + 版本号 + 明确冲突 UI |
| Cloudflare 账号锁定 | CLI 支持导出完整 D1 / R2 备份 |
| 敏感信息泄漏 | wrangler secret + GH Actions secret + 提交前扫描（`gitleaks`） |
| 主题新增组件忘记接 token | Lint 规则禁止硬编码颜色；CI 校验 |

## 七、网络安全要求（强制）

> 网络安全为**一等公民**，不是事后补丁。以下任何一条不满足即视为**功能未完成**，CI 会有对应校验。

### 7.1 传输层
- [ ] 全站强制 HTTPS；Worker 下发 `Strict-Transport-Security: max-age=63072000; includeSubDomains; preload`
- [ ] 所有 API 仅接受 TLS 1.2+
- [ ] WebSocket / Server-Sent Events 使用 `wss://`

### 7.2 认证与会话
- [ ] 密码：**argon2id** 哈希（m=64MB, t=3, p=4），禁止 MD5 / SHA1 / bcrypt-rounds<12
- [ ] Session token 存 **KV**，默认 TTL 7 天，滑动刷新
- [ ] 会话 Cookie：`HttpOnly; Secure; SameSite=Lax; Path=/; __Host- 前缀`
- [ ] OAuth：遵循 PKCE；state 参数严格校验；nonce 用于 OIDC
- [ ] 失败登录：同 IP / 同账号 5 次 / 15 分钟 → 指数退避；10 次 → 短时封锁并通知邮件
- [ ] 设备配对码：一次性、15 分钟过期、仅在登录设备显示

### 7.3 授权与多租户
- [ ] **所有**数据库查询强制按 `user_id` 过滤（Drizzle 中间件校验）
- [ ] 管理员操作走独立 `admin` 作用域 token；普通 session 永远无法调用管理员 API
- [ ] 重要管理动作（重置密码、删除用户、改登录方式）需**二次验证**（邮件一次性码）

### 7.4 输入与输出
- [ ] 所有请求体用 **Zod schema** 校验；不通过直接 400
- [ ] 输出 Markdown 渲染前走 **DOMPurify** 白名单
- [ ] 禁止服务端字符串拼接 SQL；一律 Drizzle ORM 参数化
- [ ] 文件上传：MIME + magic bytes 双重校验；大小 ≤ 20MB；拒绝 `.exe` / `.html` / `.svg`（SVG 默认拒绝，除非白名单净化）

### 7.5 CSRF / XSS / 点击劫持
- [ ] API 采用 JWT in `Authorization` 头（非 Cookie）或 Cookie + **Double-submit CSRF token**
- [ ] Worker 下发严格 CSP：
  ```
  default-src 'self';
  script-src 'self' 'wasm-unsafe-eval';
  style-src 'self' 'unsafe-inline';
  img-src 'self' data: https:;
  connect-src 'self' https://*.kairo-api.<domain>;
  frame-ancestors 'none';
  base-uri 'self';
  ```
- [ ] `X-Content-Type-Options: nosniff`、`X-Frame-Options: DENY`、`Referrer-Policy: strict-origin-when-cross-origin`、`Permissions-Policy` 最小授权

### 7.6 速率限制与滥用防护
- [ ] 未登录 API：每 IP 60 req/min
- [ ] 已登录 API：每用户 600 req/min
- [ ] 登录 / 注册 / 发邮件：每 IP 10 req/min
- [ ] 基于 Cloudflare Turnstile（免费 CAPTCHA）接入注册与忘记密码
- [ ] 管理员可配置 **IP / 地区 白名单或黑名单**（基于 `cf.country` / `cf.connectingIp`）

### 7.7 加密与密钥
- [ ] 所有密钥走 `wrangler secret`，绝不入库 / 不进 `wrangler.toml`
- [ ] VAPID、OAuth Secret、SMTP 凭证、R2 Access Key 全部 secret 管理
- [ ] R2 预签名 URL：TTL ≤ 10 分钟，仅允许用户自己的资源前缀
- [ ] 敏感字段（邮箱、IP）在日志中脱敏

### 7.8 CI / 供应链安全
- [ ] `gitleaks` 在 pre-commit + CI 扫描敏感字符串
- [ ] `pnpm audit --prod` 出现 `high` 及以上阻断合并
- [ ] 启用 GitHub **Dependabot** + **CodeQL**
- [ ] 生产构建产物 **SBOM**（CycloneDX）随 Release 发布
- [ ] 锁文件 `pnpm-lock.yaml` 必须提交；禁止 `--no-frozen-lockfile`
- [ ] GitHub Actions：最小权限 `permissions:`、OIDC 对 Cloudflare 认证（可选），禁止 `pull_request_target` 误用

### 7.9 日志、审计与响应
- [ ] 管理员操作全量审计日志，写入独立 D1 表，管理员自己**不可修改**
- [ ] 敏感错误（500）上报结构化日志到 Workers Logpush，保留 30 天
- [ ] 提供 `SECURITY.md` 与漏洞上报邮箱

### 7.10 客户端
- [ ] 移动端：`android:allowBackup="false"`、禁用明文流量、证书固定（Pinning，可选）
- [ ] 桌面端 Tauri：启用 `CSP`、禁用 `dangerousRemoteDomainIpcAccess`、allowlist 最小化
- [ ] 浏览器端：Service Worker 只缓存 `self` 来源；敏感字段不落 IndexedDB 明文（密码/token 仅在 Memory）
