# Security Policy

## 报告漏洞

请勿在公开 Issue 中报告安全漏洞。

通过邮件联系：`security@<your-domain>` （由管理员在部署时配置）
请附上：
- 漏洞描述与复现步骤
- 受影响版本
- 建议修复方案（可选）

我们承诺在 48 小时内首次回应，在 30 天内发布修复。

## 受支持版本

| 版本 | 状态 |
| --- | --- |
| `main` | ✅ 活跃开发 |
| `< 0.1` | ❌ 不支持 |
